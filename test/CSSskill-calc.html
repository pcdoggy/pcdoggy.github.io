calc()

The calc() CSS function can be used anywhere a <length>, <frequency>, <angle>, <time>, <number>, or <integer> is required.
With calc(), you can perform calculations to determine CSS property values.

引入

   /* property: calc(expression) */
   width: calc(100% - 80px);

Formal syntax

calc( <calc-sum> )
where 
<calc-sum> = <calc-product> [ [ '+' | '-' ] <calc-product> ]*
where 
<calc-product> = <calc-value> [ '*' <calc-value> | '/' <number> ]*
where 
<calc-value> = <number> | <dimension> | <percentage> | ( <calc-sum> )

例子

This is a banner! 
   Positioning an object on screen with a margin
   .banner {
	  position: absolute;
	  left: 5%;                 /* fallback for browsers without support for calc() */
	  left: calc(40px);
	  width: 90%;               /* fallback for browsers without support for calc() */
	  width: calc(100% - 80px);
	  border: solid black 1px;
	  box-shadow: 1px 2px;
	  background-color: yellow;
	  padding: 6px;
	  text-align: center;
	  box-sizing: border-box;
	}
   Automatically sizing form fields to fit their container
   input {
	  padding: 2px;
	  display: block;
	  width: 98%;               /* fallback for browsers without support for calc() */
	  width: calc(100% - 1em);
	}
	
	#formbox {
	  width: 130px;             /* fallback for browsers without support for calc() */
	  width: calc(100% / 6);
	  border: 1px solid black;
	  padding: 4px;
	}
   Nested calc() with CSS Variables
   .foo {
	  --widthA: 100px;
	  --widthB: calc(var(--widthA) / 2);
	  --widthC: calc(var(--widthB) / 2);
	  width: var(--widthC);
	}
